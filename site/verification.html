Maximum Drawdown

10% hard limit. Exceeding this threshold at any point results in immediate Challenge failure. No exceptions.

Challenge Rule Framework

The US100 Validation Challenge is evaluated on procedural compliance, not financial outcome alone. Before accessing Challenge Status, you must acknowledge and accept the following rule framework.

Session Window Trades may only be opened within the permitted challenge execution window.
Maximum Risk Risk per trade must not exceed the defined maximum allocation threshold.
Stop Loss Requirement Every trade must have a stop loss defined at entry. Trades without stop loss are treated as hard breaches.
Trade Documentation Each trade must include valid documentation (entry, exit, screenshot, notes). Incomplete trades may be invalidated.
Framework Compliance Each trade must comply with the defined execution framework. Non-framework trades may be invalidated or treated as breaches.
Invalidation vs Disqualification Some violations invalidate a single trade (it does not count). Others disqualify the entire challenge immediately and permanently.
